# Changelog — Template de spectacle

## 1.0.0 — 2026-04-15

Version initiale du template.

**Fonctionnalités couvertes :**
- Scènes `WAITING`, `PROPOSAL`, `PROMO`
- Leaderboard automatique (`hasPoints: true`)
- Thème visuel (`theme.primaryColor`, `backgroundColor`, `textColor`)
- CSS personnalisé écran et mobile (`customCss`, `customMobileCss`)
- Assets (images, audio) avec placeholder `{{asset:ID}}`
- Fond de scène par scène (`uiOverrides.backgroundAsset`)
- HTML custom par scène (`uiOverrides.customHtml`)
- Propositions avec presets et label configurable (`presetDisplayLabel`)
- Scène PROMO avec lignes cliquables (`{text, url}`)
